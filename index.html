TECNOLOGIAS

Tecnologias de trabalho para aplicações web.

HTML

HTML (Linguagem de Marcação de HiperTexto) é o bloco de construção básico da web. Define o significado e a estrutura e corpo do conteúdo exibido na web. Junto ao HTML, outras tecnologias são usadas para descrever a aparência/apresentação e a funcionalidade de uma página estática ou dinâmica da web.

CSS

CSS é uma linguagem de estilo usada para descrever a apresentação de um documento escrito em HTML ou em XML (incluindo várias linguagens em XML como SVG, MathML ou XHTML). O CSS descreve como elementos são mostrados na tela e é amplamente usados para estilizar páginas e layouts da web.

JS

Javascript é a linguagem principal para desenvolver na web, baseada em protótipos, multi-paradigma e dinâmica, suportando estilos de orientação a objetos, imperativos e declarativos.Toda interatividade da web está baseada na linguagem que possui também funcionalidades extraordinárias no servidor com o NODEJS.

SASS

O SASS é uma linguagem de extensão do CSS são folhas de estilo com uma sintaxe incrível. Seu objetivo é tornar o processo de desenvolvimento mais simples e eficiente com mais versatilidade, sendo um formato baseado em CSS cujo propósito é tornar obsoletos os problemas de desenvolvimento de frontend.
